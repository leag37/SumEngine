1 /**************************************************************************************************
2 * Title: SumDictionary.h
3 * Author: Gael Huber
4 * Description: Defines a templated dictionary. The dictionary is implemented in the form of a
5 * red-black tree. Red-black trees have the following properties:
6 * Property 1: A node is either red or black
7 * Property 2: The root is black
8 * Property 3: All leaves are the same color as the root
9 * Property 4: Both children of every red node are black
10 * Property 5: Every simple path from a given node to any of its descendant leaves contains the
11 * same number of black nodes.
